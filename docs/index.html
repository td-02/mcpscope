// Architecture

ZERO TOUCH.
FULL SIGNAL.

  • PROXY Sits between agent and server. Forwards all traffic unmodified. Your server never knows it's there.
  • STORE Every tool call - args, result, latency, error - persisted to SQLite locally. Queryable, exportable, auditable.
  • EXPORT OTLP gRPC to any compatible backend. Grafana, Jaeger, Honeycomb. One env var to connect.
  • SCHEMA Snapshot your server's tool registry to JSON. Diff any two snapshots. Exit code 1 on breaking change. Block the PR.
